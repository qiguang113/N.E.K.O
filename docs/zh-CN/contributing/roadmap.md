# 路线图

## 版本计划

### v0.7 — Agent 系统（目标：2026 年 2 月）
- 完成 Agent 相关功能
- MCP、Computer Use、Browser Use 集成

### v0.8 — 记忆系统（目标：2026 年 3 月）
- 增强记忆管理
- 类 OpenClaw 功能
- 改进记忆压缩与召回

### v0.9 — 多平台（目标：2026 年 4 月）
- Linux 支持改进
- 移动平台探索
- N.E.K.O. Network 启动（AI 社交功能）

### v1.0 — 自定义 AI（目标：2026 年 6 月）
- 过渡到自训练模型
- 专用 Agent 系统
- 减少对第三方模型服务商的依赖

## 长期愿景

### 第一阶段：创意工坊（当前）
Steam 上的核心引擎，配合创意工坊支持共享自定义内容。

### 第二阶段：独立平台
独立应用和网站，拥有更丰富的 UGC 社区以及 AI 原生游戏生态系统。

### 第三阶段：N.E.K.O. Network
自主 AI 社交化——角色拥有自己的"意识"，能够交流、组建群组，并在模拟社交媒体上发帖。
