# 部署概述

N.E.K.O. 可以根据您的环境和需求通过多种方式部署。

| 方式 | 适用场景 | 平台 |
|------|----------|------|
| [Docker](/deployment/docker) | 生产环境、服务器、无界面运行 | Linux、macOS |
| [手动搭建](/deployment/manual) | 开发、自定义 | 所有平台 |
| [Windows 可执行文件](/deployment/windows-exe) | 终端用户 | Windows |

## 最低要求

- **CPU**：2 核以上
- **内存**：最低 4 GB，推荐 8 GB
- **Python**：3.11（手动搭建时需要）
- **网络**：需要互联网访问以调用 API（除非使用本地 LLM）

## 使用的端口

| 端口 | 服务 | 是否必需 |
|------|------|----------|
| 48911 | 主服务器（Web UI） | 是 |
| 48912 | 记忆服务器 | 是 |
| 48913 | 监控服务器 | 可选 |
| 48915 | Agent 服务器 | 可选 |
| 48916 | 插件服务器 | 可选 |
