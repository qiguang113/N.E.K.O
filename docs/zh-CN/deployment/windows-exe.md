# Windows 可执行文件

面向终端用户，N.E.K.O. 提供了通过 Steam 分发的独立 Windows 可执行文件。

## Steam 安装

1. 访问 [Steam 商店页面](https://store.steampowered.com/app/4099310/__NEKO/)
2. 安装 N.E.K.O.
3. 启动 `N.E.K.O.exe`
4. Web UI 会自动在默认浏览器中打开

## 一键安装包

也提供独立安装包：

1. 下载并解压归档文件
2. 运行 `N.E.K.O.exe`
3. 在 `http://localhost:48911/api_key` 配置你的 API 密钥

## 桌面版特有功能

- Steam 创意工坊集成，用于模型分享
- Steam 成就
- 从 Steam 客户端自动检测语言
- 系统托盘集成
- 通过 Steam 自动更新
