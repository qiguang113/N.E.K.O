# Windows Executable

For end users, N.E.K.O. is available as a standalone Windows executable via Steam.

## Steam installation

1. Visit the [Steam store page](https://store.steampowered.com/app/4099310/__NEKO/)
2. Install N.E.K.O.
3. Launch `N.E.K.O.exe`
4. The Web UI opens automatically in your default browser

## One-click package

A standalone package is also available:

1. Download and extract the archive
2. Run `N.E.K.O.exe`
3. Configure your API key at `http://localhost:48911/api_key`

## Features specific to the desktop build

- Steam Workshop integration for model sharing
- Steam achievements
- Automatic language detection from Steam client
- System tray integration
- Auto-update via Steam
